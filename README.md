# Reactor Dashboard
Reactor dashboard was made as a part of a hackathon hosted by integrify. The challange was to solve the following:
- Generated revenue per employee, project
- Generated profit per project
- Amount of sick leaves and annual holidays per employee
- Amount of worked hours per employee, project (billable, non-billable)
